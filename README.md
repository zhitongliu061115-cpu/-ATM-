# 🏦 百川银行 ATM 系统 (Baichuan Bank ATM System)

> 基于 Java Swing 与 Socket 通信的 C/S 架构银行自动取款机模拟系统。

## 📖 项目简介

本项目为基于Java实现的百川银行ATM系统，主要目标是模拟真实ATM自动取款机的操作流程，实现用户与银行系统之间的交互。系统由客户端与服务器端组成，双方通过Socket进行通信。系统通过文件存储（data.txt）管理账户数据，使用多线程处理并发客户端请求，确保高可用性。创新点包括邮件发送、验证码、PDF凭条生成、语音交互、数据可视化等功能，为用户和管理员提供高效的银行服务体验。

## ✨ 核心功能
这里测试核心功能

### 🖥️ 客户端 (Client)
客户端采用 CardLayout 布局，提供流畅的单窗口操作体验。

*   **🔐 安全认证**：
    *   **登录**：账号密码 + 动态图形验证码。
    *   **注册**：新用户注册需通过**邮箱验证码**验证身份。
*   **💳 核心业务**：
    *   **查询余额**：实时同步服务器数据。
    *   **存款/取款**：支持资金变动，交易成功后可生成 **PDF 交易凭条**。
    *   **转账**：支持账户间转账，交易成功后自动生成 **交易详情二维码**。
*   **🛡️ 账户安全**：
    *   支持修改密码，修改成功后自动发送**邮件通知**。
*   **🔊 交互体验**：
    *   **语音播报**：集成文字转语音 (TTS) 技术，登录及交易成功时播放个性化语音（如“账户xxx，欢迎使用”）。
    *   **多媒体反馈**：内置舒缓背景音乐 (BGM) 及模拟物理按键的点击音效。

### ⚙️ 服务器端 (Server)
服务器端采用多线程池技术，确保高并发下的稳定性。

*   **📡 连接监控**：实时显示在线客户端列表（IP、账号、连接时间）。
*   **👥 账户管理**：管理员可进行开户、销户、查询、修改余额及重置密码操作。
*   **📊 数据可视化**：基于 **JFreeChart** 生成统计图表，展示资金分布、余额区间占比等数据。
*   **📝 日志系统**：
    *   **运行日志**：记录服务器启动、关闭等系统事件。
    *   **操作日志**：详细记录客户端的每一条指令及执行结果。
    *   **异常捕获**：实时监控并显示系统运行错误。

## 🛠️ 技术栈

| 模块 | 技术/库 | 说明 |
| :--- | :--- | :--- |
| **编程语言** | Java SE (JDK 1.8+) | 项目核心语言 |
| **图形界面** | Java Swing | `CardLayout` (客户端), `JTabbedPane` (服务器端) |
| **网络通信** | Java Socket | 基于 TCP 的自定义文本协议通信 |
| **并发处理** | ExecutorService | 线程池管理客户端连接请求 |
| **数据存储** | Java I/O | 本地文件存储 (`data.txt`) |
| **图表统计** | JFreeChart | 服务器端数据可视化分析 |
| **PDF 生成** | Apache PDFBox | 生成存款/取款交易凭单 |
| **二维码** | ZXing (Google) | 生成转账记录二维码 |
| **邮件服务** | JavaMail (javax.mail) | 发送注册验证码及安全通知 |
| **语音组件** | Jacob | 文字转语音 (TTS) 功能 |
